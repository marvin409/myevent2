# Ka-Atari Event Planner

This repository contains a Vite + React event planner app.

## Deployment

This project is configured for Vercel.

### Vercel settings
- Build command: `npm run build`
- Output directory: `dist`
- Root directory: repository root
- Framework preset: `Vite` or `Other`

### Environment variables
If you need to point the app at a custom API backend, add:
- `VITE_API_BASE_URL`

Example value:
```text
https://your-api.example.com/api
```

## Notes
- `vercel.json` is included to force static-build deployment and fallback routing to `index.html`.
- `dist/` is ignored in git because it is generated by the build step.
- `package.json` now includes a Node engine requirement for Vercel build consistency.
